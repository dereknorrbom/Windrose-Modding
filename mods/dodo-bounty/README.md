# Dodo Bounty

Scaffolded Windrose mob bounty mod for `dodo-bounty`.

Scales dodo and female dodo resource quantities, including bird meat, feathers, fat, dodo eggs, and dodo head drops where those drops exist.

## Build all variants

```powershell
python ".\modding_tools\windrose_mod_cli.py" build-mod --project-dir ".\mods\dodo-bounty" --backup-first
```

## Discover matching loot tables

```powershell
python ".\modding_tools\windrose_mod_cli.py" discover-mob-loot --keyword "dodo"
```
