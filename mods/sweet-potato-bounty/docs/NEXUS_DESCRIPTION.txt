[size=5][b]Sweet Potato Bounty - Yield Variants (2x / 3x / 5x / 10x)[/b][/size]
[size=3][color=#D4D4D8]Increase sweet potato harvest quantities while keeping seed behavior and vanilla drop logic intact.[/color][/size]

[size=4][b]What This Mod Does[/b][/size]
[size=3][color=#D4D4D8]Scales Min/Max quantities for wild sweet potato foliage and sweet potato crop harvest tables. Seed entries and chance/none behavior are left unchanged.[/color][/size]

[size=4][b]Available Variants[/b][/size]
[list]
[*][b]2x[/b] - Light boost
[*][b]3x[/b] - Balanced default
[*][b]5x[/b] - Faster group progression
[*][b]10x[/b] - High abundance / low grind
[/list]

[size=4][b]Important Notes[/b][/size]
[list]
[*]Only sweet potato quantity values are multiplied
[*]Seed drop behavior is unchanged
[*]If an entry is 0/0 in vanilla, it remains 0/0
[*]Use only one variant `.pak` at a time
[/list]

[size=4][b]Install[/b][/size]
[list=1]
[*]Choose one variant `.pak`
[*]Install it to the correct folder for how you are playing
[*]Launch game
[/list]

[size=4][b]Install Paths[/b][/size]
[list]
[*][b]Single-player:[/b] [color=#D4D4D8]Windrose\R5\Content\Paks\~mods[/color]
[*][b]Multiplayer:[/b] [color=#D4D4D8]Windrose\R5\Builds\WindowsServer\R5\Content\Paks\~mods[/color]
[*][b]Dedicated Server:[/b] [color=#D4D4D8]Windrose Dedicated Server\R5\Content\Paks[/color]
[/list]

[size=3][color=#D4D4D8]For dedicated servers, you can place the `.pak` directly in the listed folder without creating a `~mods` folder.[/color][/size]
